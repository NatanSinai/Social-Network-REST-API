# Social Network REST API

REST API for a demo of a social network containing users, posts and comments.

Created by Natan Sinai and Shira Magrafta (NASH).

## To run locally

1. Create a `.env` file like the `.env.example` file with local values.
2. Run `npm i`.
3. Run `npm run dev`.
